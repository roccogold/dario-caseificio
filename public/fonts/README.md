# Fonts Directory

## Logo DARIO Fonts (Priority Order)

Il logo DARIO userà il primo font disponibile tra:

1. **Laurentian Semi Bold** (priorità 1)
2. **Garamond Premier Semibold** (priorità 2)
3. **Versina Bold** (priorità 3)

### Laurentian Semi Bold

Aggiungi i seguenti file:
- `LaurentianPro-SemiBold.woff2` (formato preferito)
- `LaurentianPro-SemiBold.woff` (fallback)
- `LaurentianPro-SemiBold.ttf` (fallback)

**Dove acquistarlo:**
- [MyFonts](https://www.myfonts.com/products/semi-bold-laurentian-345321/)
- [Monotype](https://www.monotype.com/fonts/laurentian)

### Garamond Premier Semibold

Aggiungi i seguenti file:
- `GaramondPremierPro-Semibold.woff2` (formato preferito)
- `GaramondPremierPro-Semibold.woff` (fallback)
- `GaramondPremierPro-Semibold.otf` (fallback)
- `GaramondPremierPro-Semibold.ttf` (fallback)

**Dove ottenerlo:**
- [Adobe Fonts](https://fonts.adobe.com/fonts/garamond-premier)
- [MyFonts](https://www.myfonts.com/fonts/adobe/garamond-premier/)

### Versina Bold

Aggiungi i seguenti file:
- `Versina-Bold.woff2` (formato preferito)
- `Versina-Bold.woff` (fallback)
- `Versina-Bold.otf` (fallback)
- `Versina-Bold.ttf` (fallback)

**Dove acquistarlo:**
- [MyFonts](https://www.myfonts.com/products/bold-versina-236649)
- [Latinotype](https://www.latinotype.com/fonts/versina)

### Note sulla licenza

Assicurati di avere licenze valide per l'uso web di tutti i font che intendi utilizzare nel progetto. Il sistema userà automaticamente il primo font disponibile nella lista.
